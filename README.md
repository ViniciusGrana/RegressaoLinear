# Regressão Linear: Modelagem Preditiva com Python

Este repositório contém projetos e exercícios práticos focados em **Modelagem Preditiva**, especificamente sobre **Regressão Linear**, desenvolvidos como parte da formação técnica em **Ciência de Dados** no **SENAI de Informática de São Caetano do Sul**.

## 📊 Tecnologias e Bibliotecas

O projeto utiliza o ecossistema de Machine Learning do Python:

| Biblioteca | Aplicação |
| :--- | :--- |
| **Scikit-learn** | Implementação e treinamento do modelo de Regressão Linear |
| **Matplotlib / Seaborn** | Visualização de correlações e resíduos do modelo |
| **Pandas** | Manipulação e limpeza da base de dados para modelagem |
| **Python 3** | Linguagem base para o processamento de dados |

## 🛠️ Funcionalidades e Aprendizados

O projeto demonstra o ciclo de vida básico de um modelo de Machine Learning:

*   **Análise de Correlação:** Identificação de variáveis preditoras relevantes.
*   **Treinamento do Modelo:** Implementação do algoritmo de Regressão Linear Simples e Múltipla.
*   **Avaliação de Performance:** Uso de métricas como R² e erro médio quadrático para validar o modelo.

## 🎓 Contexto Acadêmico

Este repositório é um exemplo prático de aplicação de **Estatística** e **Machine Learning** na formação em **Ciência de Dados (SENAI)**, demonstrando competência em modelagem preditiva.
